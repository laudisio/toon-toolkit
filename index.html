Privacy Policy

TOON Toolkit is a browser extension designed to help developers convert JSON to TOON format and estimate token savings for Large Language Model (LLM) usage.

Data Collection

TOON Toolkit does not collect, store, or transmit any personal data.

All processing performed by the extension happens locally in the user's browser.

Data Usage

The extension processes text (such as JSON or TOON data) only within the browser in order to:

  • Convert JSON to TOON format
  • Convert TOON to JSON
  • Estimate token usage
  • Calculate potential token savings

No user input is sent to any server.

Third-Party Services

TOON Toolkit does not use external APIs or third-party services.

The extension operates entirely offline after installation.

Storage

The extension may store minimal preferences locally in the browser, such as:

  • Theme preference (light/dark)
  • Selected model or encoding

This information is stored locally using the browser's local storage and is never transmitted externally.

Changes to This Policy

This privacy policy may be updated in the future to reflect improvements or changes to the extension.

Contact

If you have any questions about this privacy policy, please open an issue on the project's GitHub repository.
